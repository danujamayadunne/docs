## Link Blocks

Create custom links with rich previews and metadata.

<AccordionGroup>
  <Accordion title="Features" icon="link">
    Link blocks include:

    - Automatic metadata fetching
    - Custom titles and descriptions
    - Custom icons
    - Schedule the block
    - Lock the block
    - Prioritize blocks with animations
  </Accordion>
  <Accordion title="Adding Links" icon="plus">
    To add a link block:

    1. Click "Add New Block"
    2. Choose "Block (Link)"
    3. Enter the URL
    4. Customize the preview
    5. Add a description (optional)
    6. Save your link
  </Accordion>
</AccordionGroup>

## Summery

<Steps>
  <Step title="Access Block Editor">
    Go to [blocks](https://ceynk.link/blocks) in dashboard and click "Add New Block"
  </Step>
  <Step title="Choose Block Type">
    Select the type of block you want to add:

    - Link block for URLs
    - Music block for tracks
    - Spotify block for playlists
    - YouTube block for videos
    - And more...
  </Step>
  <Step title="Add Content">
    Enter the required information:

    - URLs
    - Descriptions
    - Images
    - Custom text
  </Step>
  <Step title="Customize">
    Personalize your block:

    - Edit appearance
    - Add descriptions
    - Choose icons
  </Step>
  <Step title="Save and Preview">
    Save your block and preview how it looks on your profile. You can always edit it later.
  </Step>
</Steps>
