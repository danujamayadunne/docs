## Create an account

Learn how to set up your Ceynk profile and start sharing your links.

### Setup and Configuration

<AccordionGroup>
  <Accordion title="Create your Ceynk account" icon="user">
    To get started with Ceynk, you'll need to create an account. Visit our [signup](https://ceynk.link/signup) page and follow these steps:

    1. Enter your name
    2. Enter your email address
    3. Choose a secure password
    4. Click Create an account

    OR

    You can create a ceynk account by signup with gogole account.
  </Accordion>
  <Accordion title="Complete the onboarding process" icon="setting">
    Follow these steps to set up your Ceynk profile:

    1. Choose your unique username (ceynk.link/username)
    2. Select your creator type (Creator, Brand/Business, Student, or Personal)
    3. Add your profile details:
       - Upload a profile picture
       - Enter your display name
       - Write a compelling bio
    4. Add your social media links:
       - Add links to platforms like Instagram, Facebook, Twitter, etc.
       - Add your music streaming profiles (Spotify, Apple Music, etc.)
    5. Create your first links:
       - Add URLs with automatic metadata fetching
       - Customize link titles and descriptions
       - Add streaming links for music content
    6. Choose your plan:
       - Start with the free plan
       - Or try Pro features with a 30-day free trial
  </Accordion>
</AccordionGroup>

### Add Your Links

<AccordionGroup>
  <Accordion title="Add your first link" icon="link">
    Adding links to your Ceynk profile is simple:

    1. Go to [links](https://ceynk.link/links) in dashboard.
    2. Click "Add New Block" button
    3. Select "Block"
    4. Enter the URL and title
    5. Add a description (optional)
    6. Choose a custom icon (optional)
    7. Save your link
  </Accordion>
  <Accordion title="Organize your links" icon="folder">
    Keep your profile organized:

    - Create headers for different types of links
    - Drag and drop to reorder
    - Set link visibility preferences
    - Schedule link updates
  </Accordion>
</AccordionGroup>
